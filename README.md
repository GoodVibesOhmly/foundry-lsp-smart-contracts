### Installation

```bash
$ git clone https://github.com/YamenMerhi/foundry-lsp-smart-contracts.git
$ cd ./foundry-lsp-smart-contracts
$ forge install
```

### Usage
For compiling:
```bash
$ forge build
```

For testing:
```bash
$ forge test
```
